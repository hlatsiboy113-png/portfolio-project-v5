# Developer Portfolio

A responsive portfolio website designed to showcase my projects, technical skills, and creative work through a modern, accessible user experience.

## Features

- Responsive Bootstrap layout
- Interactive project carousel
- Photography showcase
- Animated UI components
- Contact form
- Accessible navigation

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## What I Learned

This project strengthened my understanding of responsive design, component architecture, animation, accessibility, and creating a cohesive personal brand.
